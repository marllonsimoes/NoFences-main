To install nofences services

from vstudio go to tools -> command line -> developer command prompt, navigate to bin/[Debug/Release] folder, type `installutil NoFencesService.exe`

To uninstall nofences services

from vstudio go to tools -> command line -> developer command prompt, navigate to bin/[Debug/Release] folder, type `installutil /u NoFencesService.exe`
